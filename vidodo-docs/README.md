# vidodo-docs 阅读导航

本目录继续保留现有编号文件在根目录，避免打断已有文档编号、相互引用和外部链接。

为提升阅读效率，额外提供以下分组目录作为阅读索引：

- `01-总览与定位/`：产品定位、边界与整体方向
- `02-协议与模型/`：事件协议、DSL、IR、trace、patch、CLI/MCP 模型
- `03-架构与运行时/`：技术栈、MVP 架构、模块设计、运行时、视觉后端与音频装置后端设计
- `04-测试与工程执行/`：测试、审计、任务卡、工作流与 Schema 清单

建议阅读方式：

1. 先从 `00-视听系统产品包总览.md` 建立整体认知。
2. 再按上面的分组目录选择对应阅读路径。
3. 实际编码时仍以根目录中的编号正文文件为准。
