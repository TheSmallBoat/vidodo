# vidodo-docs 阅读导航

本目录中的编号正文文件现已按主题归类到不同子文件夹中。

编号本身保持不变，用于维持文档序号体系；物理路径则按阅读主题组织。

为提升阅读效率，额外提供以下分组目录作为阅读索引：

- `01-总览与定位/`：产品定位、边界与整体方向
- `02-协议与模型/`：事件协议、DSL、IR、trace、patch、CLI/MCP 模型

- `04-测试与工程执行/`：测试、审计、任务卡、工作流与 Schema 清单

建议阅读方式：

1. 先从 `01-总览与定位/00-视听系统产品包总览.md` 建立整体认知。
2. 再按上面的分组目录选择对应阅读路径。
3. 实际编码时，以各子文件夹中的编号正文文件为准。
